# Hero Section: Text-Text-Image

A modern, responsive hero section featuring two text columns and a full-width image placeholder.

## Features
- **Responsive Layout**: Two-column text layout on desktop, single column on mobile.
- **Modern Animations**: Smooth hover and active scale animations on buttons using Tailwind CSS.
- **Clean Design**: Minimalist aesthetic with focus on typography and clear CTA buttons.
- **Placeholder Ready**: Includes a styled placeholder for the main hero image.

## Usage
Simply import and use the component in your React application:

```tsx
import Hero from './heros/text-text-image/Hero';

function App() {
  return (
    <div className="container mx-auto">
      <Hero />
    </div>
  );
}
```

## Dependencies
- **React**: Core library.
- **Tailwind CSS**: For styling and animations.
